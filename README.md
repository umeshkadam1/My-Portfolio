🚀 Personal Portfolio Website

A modern, responsive personal portfolio website built with React to showcase my projects, skills, and experience as a frontend developer. The site focuses on clean design, performance, and accessibility, with a dark, tech-inspired theme.

✨ Features

Modern dark-themed UI with a clean and minimal layout

Fully responsive across desktop, tablet, and mobile devices

Smooth scrolling and subtle animations

Dedicated sections for About, Projects, Skills, Education, and Contact

SEO-friendly structure with optimized meta tags

Performance-focused (optimized images, LCP improvements)

Light/Dark mode toggle

Easy-to-maintain component-based architecture

🛠 Tech Stack

Frontend

React JS (Vite)

JavaScript (ES6+)

HTML5

CSS3

Other Tools

Git & GitHub

Lighthouse (performance & SEO checks)

📂 Project Structure
src/
├── assets/        # Images and static assets
├── components/    # Reusable UI components
├── context/       # Theme context
├── styles/        # Styling components
├── utils/         # Static data (projects, skills, etc.)
├── App.jsx
├── main.jsx

⚙️ Getting Started

Follow these steps to run the project locally:

# Clone the repository
git clone https://github.com/umeshkadam1/My-Portfolio.git

# Navigate into the project folder
cd your-portfolio-repo

# Install dependencies
npm install

# Start the development server
npm run dev


The app will run on http://localhost:5173/ by default.
